# Task 2: Apple Stock Price Prediction

## Objective

Predict the next day's closing stock price using historical Apple stock market data.

## Dataset

Apple Stock Data

## Features Used

* Open
* High
* Low
* Volume

## Target

* Next Day Close Price

## Model

* Linear Regression

## Evaluation Metrics

* MAE
* RMSE
* R² Score

## Visualizations

* Closing Price Trend
* Correlation Heatmap
* Actual vs Predicted Plot
* Scatter Plot

## Conclusion

The model successfully learned the relationship between stock features and future closing prices and produced accurate predictions.
